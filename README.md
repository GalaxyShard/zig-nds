# zig-nds

## Getting started

- [zig](https://ziglang.org/download/)
- [devkitPro](https://devkitpro.org/wiki/Getting_Started)

```
pacman -S nds-dev
git clone https://github.com/galaxyshard/zig-nds
cd zig-nds/
zig build
```
